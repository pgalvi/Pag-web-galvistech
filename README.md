# Galvis Tech Solutions

## Sitio Web de Alta Gama para Agencia Digital

### 🎨 Diseño
- **Estética**: Premium, sofisticada, tecnológica
- **Paleta de Colores**: 
  - Fondo: Deep Navy (#0A1628)
  - Acentos: Gold (#D4AF37) y Neon Blue (#00D9FF)
- **Tipografía**: Inter (body) y Montserrat (headings)
- **Efectos**: Glassmorphism, animaciones sutiles, gradientes

### 🚀 Características
- Landing page de una sola página (SPA)
- Diseño 100% responsivo (Mobile First)
- Animaciones scroll-triggered
- Accesibilidad WCAG AAA
- SEO-optimizado semánticamente
- Performance optimizado

### 📁 Estructura del Proyecto
```
galvis-tech-solutions/
├── index.html          # Página principal
├── css/
│   ├── styles.css      # Sistema de diseño principal
│   └── animations.css  # Animaciones y transiciones
├── js/
│   └── main.js         # Interacciones y scroll effects
├── assets/
│   ├── icons/          # Iconos SVG
│   └── images/         # Imágenes y backgrounds
└── README.md           # Este archivo
```

### 🎯 Secciones
1. **Hero**: Headline impactante con CTA premium
2. **Servicios**: 3 tarjetas con glassmorphism
3. **Por Qué Elegirnos**: 3 diferenciadores clave
4. **Footer**: Links, social media, newsletter

### ⚡ Cómo Ejecutar
1. Abre `index.html` en tu navegador moderno
2. No requiere build process ni dependencias
3. Funciona offline (fuentes de Google Fonts se cargan online)

### 🌐 Compatibilidad
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- Mobile browsers (iOS Safari, Chrome Mobile)

### 📝 Recomendaciones
- Configurar este directorio como workspace activo
- Personalizar enlaces sociales en el footer
- Añadir imágenes propias en `assets/images/`
- Configurar formulario de newsletter con backend
